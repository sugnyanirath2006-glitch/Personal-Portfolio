# 🌐 Personal Portfolio – Sugnyani Rath

A simple and responsive **personal portfolio website** built using **HTML, CSS, and JavaScript**.  
This website showcases my skills, projects, and ways to connect with me.  
Designed to be clean, minimal, and professional for showcasing my work online.

---

## ✨ Features
- Multi-page portfolio (Home, About, Projects, Contact)
- Clean and modern UI
- Responsive layout
- Separate project cards
- Integrated link to my Python Task Manager project on GitHub
- Hosted via GitHub Pages

---

## 📄 Pages Included

| Page | Purpose |
|------|---------|
| **Home** | Introduction + profile section |
| **About** | Skills, background, and short bio |
| **Projects** | Portfolio + Python Task Manager project link |
| **Contact** | Simple message form UI (static) |
